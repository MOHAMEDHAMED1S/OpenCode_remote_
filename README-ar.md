# OpenCode Remote

[English](README.md) | العربية

تطبيق OpenCode Remote هو تطبيق مكتبي مخصص يتيح لك التحكم في "OpenCode Agent" عن بُعد وبشكل آمن من خلال دمج تقنيات Telegram Bot. يعمل التطبيق كجسر اتصال مستقر بين بيئة التطوير المحلية الخاصة بك وهاتفك المحمول، مما يسمح لك بمراقبة وإدارة مهام الوكيل برمجياً بسهولة واحترافية.

## التنزيل

**[تنزيل OpenCode Remote لنظام macOS (أحدث إصدار)](https://github.com/MOHAMEDHAMED1S/OpenCode_remote_/releases/download/v1.0/OpenCode_Remote-1.0.0-arm64.dmg)**

## المميزات الأساسية

* **التحكم عن بُعد:** إدارة ومراقبة OpenCode Agent عن بُعد من خلال اتصال مشفر وآمن عبر Telegram Bot.
* **لوحة تحكم احترافية:** واجهة مستخدم متقدمة وعالية الكثافة مصممة للمحترفين، تتيح لك إدارة الاتصالات، عرض حالة النظام، وتكوين الإعدادات بكل سلاسة.
* **العمل في الخلفية:** يعمل التطبيق بصمت في شريط المهام (System Tray) الخاص بنظام macOS، مما يضمن بقاء الاتصال نشطاً دون استهلاك مساحة في شريط التطبيقات (Dock).
* **إدارة مساحة التخزين:** أدوات مدمجة لإدارة قواعد البيانات والسجلات. ميزة التنظيف الآمن (Safe Clean) تتيح لك تفريغ مساحة التخزين بأمان دون فقدان بيانات الاتصال أو الإعدادات الحساسة.
* **تجربة النظام الأصلي:** مصمم خصيصاً لنظام macOS، ليوفر إشعارات النظام الأصلية، ميزة التشغيل التلقائي عند بدء التشغيل، وأداء محسن عالي السرعة.
* **بنية برمجية آمنة:** تم تصميم النظام مع التركيز على الخصوصية والأمان، لضمان بقاء قنوات الاتصال بين الوكيل وتطبيق تيليجرام محمية بالكامل.

## خطوات البدء

1. قم بتنزيل التطبيق باستخدام الرابط الموضح أعلاه.
2. افتح ملف الـ `.dmg` واسحب تطبيق **OpenCode Remote** إلى مجلد التطبيقات (Applications).
3. قم بتشغيل التطبيق واتبع تعليمات الإعداد الأولية في لوحة التحكم لربط الـ Telegram Bot الخاص بك.

## متطلبات النظام

* نظام تشغيل macOS 10.15 (Catalina) أو أحدث.
* معالجات Apple Silicon (M1/M2/M3) أو معالجات Intel.
