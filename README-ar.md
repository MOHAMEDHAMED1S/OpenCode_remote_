# OpenCode Remote

[English](README.md) | العربية

تطبيق OpenCode Remote هو تطبيق مكتبي مخصص يتيح لك التحكم في "OpenCode Agent" عن بُعد وبشكل آمن من خلال دمج تقنيات Telegram Bot. يعمل التطبيق كجسر تواصل بين بيئة التطوير المحلية (Local Development) وجهازك المحمول، مما يسمح لك بمراقبة وإدارة الـ Agent بكل سلاسة وسهولة.

> **تنويه:** تطبيق OpenCode Remote هو مشروع مستقل تم تطويره بواسطة شركة CodeMZ، وليس له علاقة رسمية أو ارتباط بالفريق الرسمي لتطوير OpenCode.

## المميزات الأساسية

* **التحكم عن بُعد:** إدارة ومراقبة OpenCode Agent عن بُعد من خلال اتصال مشفر وآمن عبر Telegram Bot.
* **لوحة تحكم احترافية:** واجهة مستخدم متقدمة وعالية الكثافة مصممة للمحترفين، تتيح لك إدارة الاتصالات، عرض حالة النظام، وتكوين الإعدادات بكل سلاسة.
* **العمل في الخلفية:** يعمل التطبيق بصمت في شريط المهام (System Tray) في نظامي (macOS & Windows)، مما يضمن بقاء اتصالك عن بُعد نشطاً دون ازدحام مساحة العمل الخاصة بك.
* **إدارة التخزين:** يتضمن أدوات مدمجة لإدارة قواعد البيانات والسجلات. تتيح لك ميزة "التنظيف الآمن" (Safe Clean) تفريغ مساحة التخزين بشكل آمن دون فقدان بيانات الاتصال أو الإعدادات الأساسية.
* **تجربة سطح مكتب أصلية:** تم بناء التطبيق باستخدام Electron لضمان أداء مستقر، وإشعارات أصلية للنظام، وإمكانية التشغيل التلقائي عند بدء التشغيل.
* **بنية برمجية آمنة:** تم تصميم النظام مع التركيز على الخصوصية والأمان، لضمان بقاء قنوات الاتصال بين الوكيل وتطبيق تيليجرام محمية بالكامل.

## كيفية الاستخدام (دليل خطوة بخطوة)

### 1. التثبيت
* **الماك (macOS):** قم بتنزيل ملف الـ `.dmg` وافتحه، ثم اسحب تطبيق **OpenCode Remote** إلى مجلد التطبيقات (Applications).
* **الويندوز (Windows):** قم بتنزيل ملف الـ `.exe` وقم بتشغيله مباشرة (النسخة Portable محمولة وتعمل فوراً بدون تثبيت).

### 2. الإعداد الأولي (ربط بوت تيليجرام)
1. افتح التطبيق. ستظهر لك شاشة الإعداد الأولية.
2. افتح تطبيق Telegram وابحث عن **BotFather** (`@BotFather`).
3. أرسل أمر `/newbot`، اختر اسماً للبوت، ثم انسخ **رمز الـ HTTP API Token**.
4. الصق الرمز (Token) في تطبيق OpenCode Remote واضغط على **Connect**.
5. بمجرد الاتصال، سيفتح التطبيق لوحة التحكم الرئيسية (Dashboard) ويتم تصغيره للعمل في شريط المهام.

### 3. حقن كود الـ Agent (Injecting)
لكي يتمكن بوت تيليجرام من التحكم في وكيل OpenCode المحلي الخاص بك، تحتاج إلى تشغيل سكريبت الربط بداخل تطبيق OpenCode:
1. افتح **لوحة التحكم (Dashboard)** الخاصة بالتطبيق من شريط المهام.
2. من القائمة الجانبية، انقر على **"Inject Guide"** لعرض الـ Server URL والـ WS Token الخاصين بك.
3. قم بنسخ السكريبت البرمجي الموضح في الشاشة.
4. افتح تطبيق **OpenCode** الأساسي على جهازك.
5. افتح أدوات المطور (DevTools) بالضغط على `Cmd+Option+I` في الماك أو `Ctrl+Shift+I` في الويندوز، واذهب إلى تبويب **Console**.
6. إذا ظهر لك تحذير أمني يمنع اللصق، اكتب `allow pasting` واضغط Enter.
7. الصق السكريبت في الكونسول واضغط Enter. سيُطلب منك إدخال الـ Server URL والـ Token، قم بإدخالهم ليتم الربط فوراً!

### 4. التحكم عن بُعد عبر Telegram
1. اذهب إلى البوت الخاص بك الذي أنشأته للتو في Telegram.
2. أرسل أمر `/start`.
3. اذهب إلى تطبيق OpenCode Remote > **تبويب المستخدمين (Users)** وقم بالموافقة (Approve) على حسابك.
4. مبروك! يمكنك الآن الدردشة مع الـ Agent، وإرسال الأوامر، ومراقبة حالته مباشرة من تيليجرام!

## متطلبات النظام

* **الماك (macOS):** نظام macOS 10.15 (Catalina) أو أحدث. يدعم معالجات Apple Silicon (ARM64) ومعالجات Intel.
* **الويندوز (Windows):** نظام Windows 10 أو Windows 11 (نسخة 64-bit).
